## Quick Examples

### Hello World

```joel
[Interpreted]

fn main() {
  print("Hello, JOEL!")
}
```

### Systems Programming

```joel
[Compiled]

fn fibonacci(n: int) -> int {
  if n <= 1 {
    return n
  }
  return fibonacci(n - 1) + fibonacci(n - 2)
}

fn main() {
  print(fibonacci(10))  // 55
}
```

### Smart Contracts

```joel
[Compiled]
#target: EVM

actor Wallet {
  balance: int = 0
  
  fn deposit(amount: int) {
    self.balance += amount
  }
  
  fn withdraw(amount: int) {
    if amount <= self.balance {
      self.balance -= amount
    }
  }
}
```

### UI Components

```joel
[Compiled]
#target: WASM

component Button {
  label: str
  onClick: fn()
  
  fn render() -> Element {
    return <button onClick={self.onClick}>
      {self.label}
    </button>
  }
}
```
