# Built-in Functions

JOEL provides built-in functions for common operations.

## Print Functions

### print

Print values to standard output.

```joel
print("Hello")              # Hello
print("x =", 10)            # x = 10
print("Sum:", 5 + 3)        # Sum: 8
```

## Range Functions

### range

Generate a range of numbers.

```joel
# Single argument: 0 to n-1
for i in range(5) {
  print(i)  # 0, 1, 2, 3, 4
}

# Two arguments: start to end-1
for i in range(2, 6) {
  print(i)  # 2, 3, 4, 5
}
```

## Examples

### Using Built-ins

```joel
[Interpreted]

fn main() {
  # Print
  print("Counting:")
  
  # Range
  for i in range(0, 5) {
    print("  ", i)
  }
  
  # Output:
  # Counting:
  #   0
  #   1
  #   2
  #   3
  #   4
}

main()
```
