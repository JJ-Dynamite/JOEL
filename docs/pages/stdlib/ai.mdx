# AI Module

The AI module provides machine learning and AI capabilities.

## Tensor Operations

```joel
[Interpreted]

import ai

# Create tensors
let t1 = ai.tensor([[1, 2], [3, 4]], dtype:"f32")
let t2 = ai.tensor([[5, 6], [7, 8]], dtype:"f32")

# Operations
let result = ai.matmul(t1, t2)
let sum = ai.sum(t1)
let mean = ai.mean(t1)
```
